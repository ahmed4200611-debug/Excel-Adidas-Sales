# Adidas Sales Dashboard

## Overview
This project is an interactive Adidas Sales Dashboard built using Microsoft Excel Pivot Tables and Pivot Charts. The dashboard provides insights into sales performance, operating profit, sales methods, product categories, quarterly trends, and monthly sales analysis.

## Features
- Sales analysis by method (In-store, Online, Outlet)
- Quarterly sales performance tracking
- Monthly sales trend analysis
- Product-wise profit analysis
- Interactive filters (Slicers)
- Dynamic Pivot Charts and Pivot Tables
- Clean and user-friendly dashboard design


## Key Metrics
- Total Sales
- Operating Profit
- Sales by Product
- Sales by Quarter
- Sales by Month
- Sales by Sales Method

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## Insights
- Identify the best-performing sales channels.
- Track seasonal sales trends.
- Analyze profitability across product categories.
- Compare quarterly sales performance.
